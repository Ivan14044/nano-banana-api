# Быстрый старт

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python main.py
```

## Первое использование

1. При первом запуске API ключ уже установлен по умолчанию
2. При необходимости вы можете изменить его в верхней панели
3. Нажмите "Проверить баланс" для проверки доступных кредитов

## Основные функции

### Генерация изображений

1. Вкладка "Генерация"
2. Выберите режим: одиночная или пакетная генерация
3. Введите описание (для пакетной - каждое с новой строки)
4. Выберите модель и разрешение
5. Нажмите "Сгенерировать"

### Редактирование

1. Вкладка "Редактирование"
2. Загрузите изображение
3. Введите инструкции по редактированию
4. Нажмите "Редактировать"

### Комбинирование

1. Вкладка "Комбинирование"
2. Добавьте 2-8 изображений
3. Введите инструкции
4. Нажмите "Объединить"

### Галерея

1. Вкладка "Галерея"
2. Просматривайте все сгенерированные изображения
3. Используйте поиск и фильтры
4. Просматривайте и скачивайте изображения

## Пакетная генерация

Для пакетной генерации:
1. Выберите "Пакетная генерация"
2. Введите несколько промптов, каждое с новой строки
3. Все изображения будут сгенерированы последовательно
4. Каждое изображение сохранится в галерею

## Примечания

- Изображения сохраняются в папку `data/images/`
- История сохраняется в базу данных `data/history.db`
- Настройки сохраняются в `~/.nanobanana_pro/config.json`
