# История изменений

## [1.1.0] - Обновление под официальную документацию API

### Изменения

- ✅ **Полностью переписан API клиент** согласно официальной документации NanoBanana API
- ✅ **Асинхронная обработка задач** - теперь используется polling вместо прямых ответов
- ✅ **Правильные эндпоинты**:
  - `/api/v1/nanobanana/generate` - обычная генерация
  - `/api/v1/nanobanana/generate-pro` - Pro генерация
  - `/api/v1/nanobanana/record-info` - проверка статуса задачи
  - `/api/v1/common/credit` - проверка баланса

### Новые возможности

- ✅ **Polling механизм** - автоматическое опрашивание статуса задач каждые 3 секунды
- ✅ **Загрузка изображений** - поддержка загрузки изображений на публичные URL для редактирования и комбинирования
- ✅ **Правильная обработка ответов** - корректная обработка формата ответов API

### Исправления

- ✅ Исправлены URL эндпоинтов
- ✅ Исправлена обработка taskId
- ✅ Исправлена проверка баланса
- ✅ Добавлена поддержка aspect ratio для Pro API

### Известные ограничения

- ⚠️ Редактирование и комбинирование требуют загрузки изображений на публичные URL
- ⚠️ Используется временный хостинг tmpfiles.org для загрузки изображений
- ⚠️ Callback URL обязателен в API, но не используется (используется фиктивный URL)
